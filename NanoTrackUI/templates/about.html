About This Project

NanoTrack is a real-time small object detection and tracking interface optimized for edge devices such as the Jetson Orin NX. The purpose of this project is to demonstrate how modern computer vision models (e.g. EdgeYOLO, Norfair, ByteTrack) can be integrated into an accessible, easy-to-use tool for research and educational purposes.

Technologies Used

  • YOLO-based object detection models (fine-tuned for small objects)
  • Tracking algorithms: Norfair & ByteTrack
  • Torch / TensorRT optimization for high FPS on edge devices
  • Python + Flask backend with a modern HTML/CSS frontend

Author

This project was developed by Ulaş Sakın, an Electrical and Electronics Engineering student, as part of ongoing efforts to learn and experiment with machine learning, computer vision, and embedded systems. You can find the source code on GitHub.
